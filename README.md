# WinPars

WinPars is a modern betting platform designed for sports betting and related services.

## Purpose

This repository is used for the development and maintenance of the WinPars betting platform.

## Features

- User Authentication
- Sports Betting
- Live Betting
- Bet History
- Wallet Management
- Odds Management
- Admin Panel
- Reporting & Analytics
- API Integration

## Technologies

- PHP
- MySQL
- Bootstrap
- JavaScript
- HTML5
- CSS3
- REST API
